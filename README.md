# 🎸 Meu Portfólio de Covers de Guitarra

Este é um projeto pessoal desenvolvido como um desafio prático de Front-End para consolidar meus aprendizados em **HTML5, CSS3 e JavaScript Assíncrono/Manipulação de DOM**.

## 🚀 Funcionalidades
- Grid de vídeos totalmente responsivo usando CSS Grid.
- Player de vídeo integrado via YouTube Embed API dentro de um Modal customizado.
- Fechamento inteligente do modal via clique externo ou tecla `Escape`.
- Consumo dinâmico de dados através de `data-attributes` no HTML.

## 🛠️ Tecnologias Utilizadas
- HTML5 Semântico
- CSS3 (Flexbox, Grid, Variáveis e Efeitos de Transição)
- JavaScript Puro (Vanilla JS)
